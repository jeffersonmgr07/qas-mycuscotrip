# MyCuscoTrip - Agencia de Viajes

Sitio web para agencia de viajes especializada en tours a Cusco y Machu Picchu.

## 🚀 Características

- Diseño responsive moderno
- Sistema de búsqueda avanzada con Flatpickr
- Componentes reutilizables
- Optimizado para SEO
- Integración con WhatsApp
- Estructura escalable para futuras integraciones
